# 🎭 Madlibs Game

A simple and fun Python command-line project that takes user inputs (adjectives, nouns, verbs) and dynamically injects them into a pre-defined story template. 

## 🚀 Features
* Interactive command-line interface.
* Dynamic string manipulation using Python's formatting.
* Fun storytelling mechanism.

## 🛠️ How to Run
Make sure you have Python installed, then run the following command in your terminal:
```bash
python madlibs.py
