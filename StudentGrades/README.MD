# Student Grade Management System

A lightweight, terminal-based CRUD (Create, Read, Update, Delete) application built in Python to seamlessly manage student grades. This script utilizes a centralized dictionary structure to store data dynamically during runtime.

## 🚀 Features

*   **Add Students:** Instantly register a student along with their numeric grade.
*   **Update Records:** Easily modify existing student grades with real-time feedback.
*   **Delete Entries:** Safely remove a student's record from the system.
*   **View All Data:** Display a clean list of all registered students and their corresponding grades.
*   **Interactive Menu:** An intuitive, infinite-loop command-line interface (CLI) that keeps running until you choose to exit.
